처음으로 만드는 Web page

처음으로 만드는 HTML 문서로 작성된 Web page, Web Page 는 HTML 형식의 문서로 작성한다. HTML 문서에서 p tag 로 시작하면 하나의 문단이 시작된다. 하나의 문단은 close /p 이며 문단이 끝났음을 나타낸다. 문단 내에서는 아무리 엔터를 입력하여도 브라우저에 나타날때는 모드 무시 된다. 단 엔터를 입력하면 한칸의 빈칸이 나타난다. 문단 중간에서 여러개의 빈칸(스페이스)를 입력 하여도 한개의 빈칸만 남기고 모두 무시된다.

HTML 문서는 tag 로 감싸진 Text 로 구성되어 있다 tag 로 감싸진 문서를 작성하고 Web browser 를 통하여 확인한다

box tag, box 모양의 공간을 만들고 text 등을 표현하는 tag
